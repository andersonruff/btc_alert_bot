  BOT ALERTA DE BITCOIN
  
Uma idéa que automatiza a necessidade de abrir o computador, sites o tempo todo pra verificar o preço do BTC, esse projeto envia mensagens informando o RSI e o preço do ativo a cada uma hora, roda gratuitamente 
pelo **GitHub Actions** que disponibiliza 2000 minutos mensais, mais que o suficiente pra manter o projeto funcionanto e automatizando uma terefa, poupando tempo e fornecendo informações uteis a respeito de um 
ativo muito importante, ainda ajudando pessoas leigas que não entendem de RSI o projeto apontando sobrevenda de RSI em um periodo de 14, tendo possibilidades de adicionar mais metricas e deixar ele um verdadeiro 
assistente pessoal, obrigado e segue mais inforações do projeto.



---

## O que este bot faz

- Consulta o preço atual do BTC (BTC-USD)
- Calcula o indicador RSI (14)
- Envia uma mensagem para o Telegram
- Execução manual ou agendada
- Automatiza uma tarefe importante!!

---

## Tecnologias Utilizadas

- Python 3.11
- yfinance
- ta (Technical Analysis)
- requests
- GitHub Actions
- Telegram Bot API

OBJETIVO DO PROJETO:

- Aprender automação com GitHub Actions
- Entender consumo de APIs
- Praticar Python com dados financeiros
- Integrar Python com Telegram
